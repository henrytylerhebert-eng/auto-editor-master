# Release Checklist

Please ensure the following checks are complete before using the release workflow:

- [ ] 1. Smoke CI passes on `main`.
- [ ] 2. Release branch/tag is signed or reviewed.
- [ ] 3. Dependencies were downloaded and hash-verified.
- [ ] 4. Binary names match expected platform matrix.
- [ ] 5. Release notes identify fork status.
- [ ] 6. Upload is only allowed from protected refs.
- [ ] 7. Review [SECURITY_BUILD_NOTES.md](SECURITY_BUILD_NOTES.md).
